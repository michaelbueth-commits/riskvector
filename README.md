## 🚀 Deployment auf GitHub Pages

### Option 1: GitHub Repository
1. Erstelle neues Repository: `crisislens` oder `crisislens.github.io`
2. Lade diese Dateien hoch
3. Gehe zu Settings → Pages
4. Quelle: "Deploy from a branch"
5. Branch: `main`
6. Ordner: `/ (root)`
7. Klicke "Save"

### Option 2: Bestehendes Repository
1. Erstelle `gh-pages` Branch
2. Lade die Landing Page Dateien hoch
3. GitHub Pages automatisch unter: `https://[username].github.io/[repo]/`

## ⚙️ Konfiguration

### Domain anpassen
Falls du eine eigene Domain verwendest:
1. In `index.html` die Beta-Anmeldung anpassen:

Aktuell nutzt das Formular GitHub Issues. Alternativen:
